# 💰 Expense Tracker Android App

A modern Android Expense Tracker application built using **Kotlin**, **SQLite**, and **Material Design**. The app helps users manage their daily income and expenses, visualize financial data with charts, and keep track of transaction history through an intuitive interface.

---

## 📱 Features

### 🏠 Dashboard
- View current balance
- Income and expense summary
- Financial overview using Pie Chart
- Recent transactions

### 💳 Transaction Management
- Add Income
- Add Expense
- Edit existing transactions
- Delete transactions
- Categorize transactions
- Add notes
- Store transaction date

### 📜 Transaction History
- View all transactions
- Search transactions
- Sort transactions
- Filter transactions
- Clean RecyclerView interface

### 📊 Reports
- Income vs Expense Pie Chart
- Monthly Spending Bar Chart
- Quick Statistics
- Category Analysis
- Net Balance Summary

### 💾 Database
- SQLite Database
- CRUD Operations
- Persistent Local Storage

---

## 🛠 Tech Stack

- **Language:** Kotlin
- **IDE:** Android Studio
- **Database:** SQLite
- **UI:** XML Layouts
- **Charts:** MPAndroidChart
- **Architecture:** Activity-Based
- **Design:** Material Design Components

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/devidjoshi143-droid/Syntecxhub_ExpenseTracker.git
```

2. Open the project in Android Studio.

3. Sync Gradle.

4. Run the application on an emulator or Android device.
